# Snowmen Invasion

a lot of snowmens (from space!)
invading your home !

##### strike them back right now!

## Instructions:

- use the Arrrow Keys to move the Player
- 's' to shoot
- 'p' to take a break (not too long!)
- avoid the trees, they will push you down!
